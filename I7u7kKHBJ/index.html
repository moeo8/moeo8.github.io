Gridea添加Gitalk，Disqus评论系统

Author: 温故而知新 Date: 2021-02-21 Reading Time:3.5 mins words:820 Category: Gridea

说明

Gitalk 是一个基于 GitHub Issue 和 Preact 开发的评论插件。

特性
  • 使用 GitHub 登录
  • 支持多语言 [en, zh-CN, zh-TW, es-ES, fr, ru, de, pl, ko]
  • 支持个人或组织
  • 无干扰模式（设置 distractionFreeMode 为 true 开启）
  • 快捷键提交评论 （cmd|ctrl + enter）
  • 博客同步后，每一篇需要你手动去激活评论，别人才能评论。

配置好后，页面最终效果(https://gitalk.github.io/)。

Disqus是商业插件，可以使用免费版本（可能会有广告），用的人很多，而且注册时不需要提供手机号。

Gitalk使用

  1. 选择一个公共github存储库用于存储评论。
  2. 创建 GitHub Application，如果没有 点击这里申请，Authorization callback URL 填写当前使用插件页面的域名。
  1. 成功以后记下Client ID，Client secrets

Gitalk安装

官方的安装方式有2种，但是Gridea，可以直接配置信息输入。

注意：博客同步后，每一篇需要你手动去激活评论，也就是在评论的地方需要你使用Github先登陆一下，这样就完成了初始化，别人才能评论哟。

注册及配置 Disqus

前往 Disqus 官网 注册完成后，选择 I want to install Disqus on my site

填写自己的网站信息，输入域名即可。

![img](https://cdn.sspai.com/2020/12/25/3b68cafd48c384e9fe5ba43e89b486bf.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1 style="zoom:70%;" /)选择订阅计划（可以免费体验 30 天）

由于我的博客是使用 Gridea 配合搭建的，这里可以先不用去选择。

![img](https://cdn.sspai.com/2020/12/25/5cb43a67e0f7b043e781d6b640e5128a.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1 style="zoom:70%;" /)

直接前往 Disqus API Resources 新建一个 Application

![img](https://cdn.sspai.com/2020/12/25/095ab78b85fdbf6fee561a4f976d3e6b.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1 style="zoom:70%;" /)将基础信息完善后，点击 Redister my application 确认

基础信息完善后，进入API 信息页面，在「 Details 」中获取你的 API Keys 与 API Secrets 并复制到剪贴板和其他文本上

![img](https://cdn.sspai.com/2020/12/25/8b73a6fb00659d3c7762845ad5340aa8.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1 style="zoom:70%;" /)接着切换到「 Settings 」中 Domains ，设置域名白名单防止其他用户滥用你的 API ，还能防止某些安全隐患。

![img](https://cdn.sspai.com/2020/12/25/1bbddeff1a23ed29b92adb1d0d7d8c10.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1 style="zoom:70%;" /)信息全部完善后，直接打开已安装的 Gridea 应用找到「 远程 」中 评论配置。

输入 Disqus 创建账号时填写的 Shortname (Website Name)，同时复制 API 信息页面中的 API Keys 与 API Secrets 对应 Gridea 中的 apikey 和 api ，确认后保存即可。

点击应用左下角的「 同步 」，等待同步完成后访问自己的博客文章就可以看到文章最底部的 Disqus 评论功能。

![img](https://cdn.sspai.com/2020/12/25/1b97c194ccbcb0d6a47736e4f15202fc.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1 style="zoom:70%;" /)

Disqus 评论数据

前往 Disqus 中的 Analytics 数据中心可以查看用户在访问博客文章时的留言评论信息，可帮助博主更好地了解受众与互动内容。
