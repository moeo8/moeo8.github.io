静态博客系统比较

动态部署 WordPress、Typecho 等

资源占用大，大材小用，没有必要。

Jekyll、Hexo

可用 Travis CI 自动部署，但要本地push比较麻烦，不考虑。

Maverick

基于 Lepture 的 mistune 加入了一些扩展的 Markdown 语法。可以利用 GitHub 提供的自动化服务 GitHub Actions，直接用github书写管理。

参考文章 推荐站点

Gridea

由于各种原因，最后用上了Gridea，国人开发，现在还不是很稳定（2021.02.20），主题较少，没有插件系统，cdn不完善，但是可用，而且架不过有客户端。部署最简单。不用只为一个博客，装一堆环境。
